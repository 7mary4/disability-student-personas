# Student Support Persona Cards

A comprehensive teacher resource library covering 60 student disability profiles, grounded in Universal Design for Learning (UDL) and the Individuals with Disabilities Education Act (IDEA). Each card provides grade-band–specific guidance for Elementary (K–5), Middle School (6–8), and High School (9–12), with practical classroom observations, recommended accommodations, assistive technology, UDL strategies, IEP/504 considerations, collaboration recommendations, and family partnership tips.

**Web version:** https://fyvr.net/student-personas/
